# Descriptores de archivos

### STDIN -> 0
Entrada estándar.

Quiere la entrada de datos al programa.


### STDOUT -> 1
Salida Estándar.

Quiere decir la salida normal del programa
```bash
# Ejemplo
ls /home/user/ # Salida normal
Desktop  Documents  Downloads  Music  Pictures  Public  Templates  Videos
```

### STDERR -> 2
Error Estándar

Quiere decir la salida de errores del programa
```bash
#Ejemplo
ls /noexiste
# Salida
ls: cannot access '/hhd': No such file or directory # Stderr
```
